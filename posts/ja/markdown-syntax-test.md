## リスト

### 順序なしリスト

- リスト項目1
- リスト項目2
  - ネストされたリスト項目2.1
    - さらに深いネスト2.1.1
- リスト項目3

### 順序付きリスト

1. 最初の項目
2. 2番目の項目
   1. ネストされた最初の項目
   2. ネストされた2番目の項目

3. 3番目の項目

### タスクリスト

- [x] 完了したタスク
- [ ] 未完了のタスク
- [x] もう一つの完了したタスク


```python
# Pythonのコード例
def factorial(n):
    if n == 0:
        return 1
    else:
        return n * factorial(n-1)

print(factorial(5))
```
